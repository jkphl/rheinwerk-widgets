03 Tabs

Siehe APG: Tabs Pattern

Aufgabe

Übertrage die folgende Liste von Screenreadern in eine barrierefrei zu bediendene Tab-Navigation. Die Tab-Reiter sollen dabei die Screenreader-Namen wiedergeben, die Registerkarten die Details liefern. Sei ruhig kreativ im Übertragen der Inhalte — diese müssen nicht 1:1 so dargestellt werden, wie sie im Original zu sehen sind. Es geht ums Prinzip …

Bonus

Wenn du nach dem Lösen der Aufgabe eine zusätzliche Herausforderung suchst: Kannst du dir eine Umsetzung vorstellen, die dem Prinzip des Progressive Enhancement folgt und auch ohne JavaScript nutzbar ist? Tipp: Denke nicht zu kompliziert.

  1. JAWS (Job Access With Speech)

    • Betriebssystem: Windows
    • Beschreibung: JAWS ist einer der bekanntesten Screenreader für Windows und bietet umfangreiche Funktionen zur barrierefreien Nutzung von Computern. Er unterstützt viele Anwendungen und bietet flexible Anpassungsmöglichkeiten. Weitere Informationen findest du auf der offiziellen Website von JAWS: freedomscientific.com/jaws

  2. NVDA (NonVisual Desktop Access)

    • Betriebssystem: Windows
    • Beschreibung: NVDA ist ein kostenloser, Open-Source-Screenreader, der speziell für Windows entwickelt wurde. Er bietet eine gute Unterstützung für verschiedene Anwendungen, darunter Webbrowser, Textverarbeitungsprogramme und mehr. Weitere Informationen findest du auf der offiziellen Website von NVDA: nvaccess.org

  3. VoiceOver

    • Betriebssystem: macOS, iOS
    • Beschreibung: VoiceOver ist der integrierte Screenreader von Apple und ist auf macOS- und iOS-Geräten verfügbar. Er bietet umfassende Unterstützung für die Bedienung von Apple-Produkten, einschließlich der Nutzung von Apps, dem Internet und anderen Funktionen. Weitere Informationen findest du auf der offiziellen Apple-Website: apple.com/accessibility/voiceover

  4. TalkBack

    • Betriebssystem: Android
    • Beschreibung: TalkBack ist der integrierte Screenreader für Android-Geräte. Er ermöglicht Menschen mit Sehbeeinträchtigungen die Navigation, Interaktion und Bedienung von Android-Apps. Weitere Informationen findest du auf der offiziellen Android-Website: android.com/accessibility/talkback

  5. Orca

    • Betriebssystem: Linux
    • Beschreibung: Orca ist ein Open-Source-Screenreader, der auf Linux-Systemen verwendet werden kann. Er bietet Unterstützung für verschiedene Anwendungen und ermöglicht es Benutzern, den Bildschirm zu lesen, Eingaben zu tätigen und mehr. Weitere Informationen findest du auf der offiziellen GNOME-Website: help.gnome.org/users/orca
